# “Keyness Measures und BERTopic kombiniert: Eine Distinktivitätsanalyse von Subgenres des französischen Romans”

Repository für unseren Beitrag zur  Jahrestagung des Verbandes Digital Humanities im deutschsprachigen Raum 2026 (23.–27.02.2026), Universität Wien. 

Die Studie stellt einen hybriden Analyseansatz vor, der word-embedding-gestützte Topic-Modeling-Verfahren mit Distinktivitätsmaßen kombiniert, um Textgruppen zu vergleichen.

Projekt-Website: https://zeta-project.eu/de/
